## Setup

| Tool | Version  |
| ---- | -------- |
| Node | `v18+`   |
| Vite | `v5.0.11`|
| npm  | `v7+`    |

```
$ npm create vite@latest trustpilot-challenge -- --template react-swc
$ npm install
$ npm run dev
```
